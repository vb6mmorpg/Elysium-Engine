Bem-vindo(a) ao Server-Side!

Sim, é isso mesmo que você está pensando...
Para abrir o famoso "Server-Side" você só precisa executar esse Elysium Engine Brasil (Cliente).vbp.
Não se esqueça de que você precisa do Visual Basic 6 para executá-lo.

ATENÇÃO: Você não conseguirá rodar o servidor direto pelo VB6.
Para fazê-lo, mova o arquivo com a extensão VBP e a pasta Código para dentro da pasta do Servidor.
